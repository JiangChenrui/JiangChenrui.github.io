培养专注的习惯，学会独立思考

学习AI需要哪些技能

  • 1.编程基础

    • 熟练使用linux，git，vim等环境和工具。
    • 熟练掌握 C/C++、Python等编程语言。
    • 熟练掌握cmake，g++等编译工具。

  • 2.算法基础

    • 熟悉传统图像算法，机器学习算法。
    • 熟练跟踪并阅读行业前沿研究，复现结果。
    • 系统性熟悉深度学习理论。

  • 3.框架基础

    • 熟练掌握 Caffe、TensorFlow、pytorch等以及不断新出的开源平台。

  • 4.其它基础

    • 掌握爬虫等前后端的基础知识。
    • 了解并熟悉Cuda等GPU编程技术，了解一些移动端的硬件知识。
    • 了解并熟悉 Android、iOS 等移动端的基础知识，在项目中可能会需要使用。
